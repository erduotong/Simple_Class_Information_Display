# Simple_Daily_Desktop
[![Generic badge](https://img.shields.io/badge/编写于_Python_版本-3.11.3-blue.svg)](https://Python.org)  
>这是一个可以交互的,一般来说位于壁纸层的应用
## 适用场景
开发是为了在学校进行使用 用于自动化存放作业，课表等。减少一些日常的枯燥劳动。同时也方便进行使用
