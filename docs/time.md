# 存储时间
>位置: data/Curriculum/time.json    

一个写好的示例:
```json
{
  "l1": {
    "start": "08:30",
    "end": "09:15"
  },
  "l2": {
    "start": "09:25",
    "end": "10:10"
  },
  "l3": {
    "start": "10:20",
    "end": "11:05"
  },
  "l4": {
    "start": "11:15",
    "end": "12:00"
  },
  "l5": {
    "start": "14:00",
    "end": "14:45"
  },
  "l6": {
    "start": "14:55",
    "end": "15:40"
  },
  "l7": {
    "start": "15:55",
    "end": "16:40"
  },
  "l8": {
    "start": "16:50",
    "end": "17:35"
  },
  "早操": {
    "start": "07:40",
    "end": "08:10"
  },
  "晨读": {
    "start": "08:15",
    "end": "08:30"
  },
  "延时服务": {
    "start": "17:45",
    "end": "18:30"
  },
  "中午休息": {
    "start": "12:00",
    "end": "13:40"
  }
}
```
其中的l1-l8表示第一节课到第八节课的时间 可以任意长度 但是一定要比lessons.json中列表最大的课程数长  
程序会根据今天的lessons的课表的长度来选取 比如今天只有7节课 那么程序就会读取l1-l7的课程的时间并且写入到daily_config中  
除开l1-l8,剩下的为lesson中special的课程 并且**一定要有special内的所有课程的时间**否则会出问题
